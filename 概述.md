# 概述

出差已经很辛苦了，为了费用报销的那笔小钱钱还得贴票填单真是心力憔悴......这个软件可以极大地帮助你。

你只要把收到的电子发票（pdf文件）汇总到同一个目录中，运行一次这个程序，它就会：

1、按照财务的普遍要求，把所有发票以图像形式插入一个word文件中，每页上下排列2张，方便你打印后对折裁剪作为报销凭证。

2、财务还会要求你把增值税专用发票再打印一遍作为报税之用，所以程序会贴心地把专票再次往word文件插入一遍。

3、为了方便你填报销单，程序尽可能地把发票中的诸要素提取出来汇总入excel文件中，你可以方便地看到你能拿回多少小钱钱，这可比生命还重要。

4、word发票排版及excel汇总均按费用发生日期排序，这点很重要。有的销售几个月才想起开发票，但公司一般要求按出差顺序报销。程序尽可能地从发票中找到发生日期的线索。餐费发票上只有开票日期，但好在你和客户霍霍完一顿大餐后，只要不打算自费肯定会在结账时就要开发票的。

5、本来程序也应该具备自动填报销单的功能（只需你准备好这些理由：为何真人要去不能远程吗？为何坐飞机而不买绿皮车站票？为何住商务酒店而没有睡桥洞？为何要喝茅子？为何要唱K？......其它格式化的东西程序能自动填好）但鉴于各单位百花齐放的报销规矩和填报格式，我只能先满足自己就不放上来献丑了。你若需要这样的功能，要么自己开发，要么说点好听的来求我......
