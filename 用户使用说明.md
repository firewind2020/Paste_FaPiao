# 贴发票工具：详细功能介绍与使用说明（非专业用户版）

## 这是什么工具

这个工具是给“需要报销的人”准备的。你把电子发票 PDF 放到同一个文件夹里，运行一次程序，它会自动帮你完成两件费时的事：

1. 生成 Excel 明细表：方便你核对金额、填写报销单。
2. 生成 Word 粘贴版：每页上下 2 张发票，方便打印、裁剪、粘贴。

另外，它还会把“专用发票”在 Word 末尾再排一遍，满足常见财务场景（专票单独留存/打印）。

---

## 它能帮你做什么（核心功能）

### 1. 自动识别发票类型

支持 4 类常见单页电子发票：

1. 住宿
2. 餐饮
3. 高铁
4. 飞机

程序会根据发票内容自动判断类型，不需要你手工分类。

### 2. 自动提取关键信息

统一提取这些信息：

1. 文件名
2. 发票类型
3. 是否专票
4. 开票日期
5. 发生日期
6. 发票号码
7. 购买方名称
8. 发票金额
9. 销售方名称

并按发票类型补充额外信息：

1. 住宿：住宿天数
2. 高铁：始发站、终到站、车次、发车日期、发车时间、车厢座位号、乘客姓名
3. 飞机：航班号、始发城市、目的城市

### 3. 自动按“发生日期”排序

输出会按发生日期升序排列。这样你报销按行程整理时会更顺手。

### 4. 自动生成 Word 粘贴版

1. 每页上下排 2 张发票图片。
2. 第一轮插入全部发票。
3. 末尾换页后，专票再插入一轮。

## 使用前准备（只需一次）

### 1. 安装 Python（如果电脑还没有）

建议安装 Python 3.10 及以上版本，安装时勾选“Add Python to PATH”。

### 2. 安装依赖库

在本项目所在文件夹打开 PowerShell，执行：

```bash
pip install pandas pdfplumber pymupdf python-docx openpyxl
```

说明：

1. openpyxl 用于写 Excel。
2. 如果公司网络慢，安装可能需要几分钟。

---

## 每次使用的标准步骤

### 第一步：准备发票文件

1. 新建一个文件夹（例如“2026-04 报销发票”）。
2. 把本次要报销的 PDF 发票都放进去。
3. 确保文件是 PDF 格式，且尽量是清晰的电子发票。

建议：文件名能看懂最好，例如“餐饮_客户招待_2026-04-03.pdf”。

### 第二步：运行程序

进入项目目录后执行：

```bash
python paste_fapiao.py --input-dir "你的发票文件夹路径"
```

例如：

```bash
python paste_fapiao.py --input-dir "."
```

如果你把发票就放在当前目录，使用上面这条即可。

### 第三步：查看输出结果

默认会得到两个文件：

1. 发票信息提取结果.xlsx
2. 发票粘贴.docx

如果你只想要 Excel，不生成 Word：

```bash
python paste_fapiao.py --input-dir "." --skip-word
```

如果你想自定义输出文件名：

```bash
python paste_fapiao.py --input-dir "." --output "我的报销明细.xlsx" --word-output "我的发票粘贴版.docx"
```

## 输出文件怎么用

### Excel 明细（发票信息提取结果.xlsx）

用途：

1. 核对是否有漏票。
2. 统计本次报销金额。
3. 作为填报销单时的参考清单。

建议重点检查：

1. 发票类型是否正确。
2. 金额是否正确。
3. 发生日期是否符合你的出差时间。
4. 高铁/飞机的行程字段是否合理。

### Word 粘贴版（发票粘贴.docx）

用途：

1. 直接打印。
2. 对折/裁剪后贴到报销单。

说明：

1. 每页 2 张发票。
2. 文末有专票重复区，供财务额外留存。

## 常见问题与处理办法

### 1. 提示“未找到 PDF 文件”

原因：输入目录里没有 PDF。

处理：

1. 检查路径是否正确。
2. 确认发票扩展名是 .pdf。

### 2. 提示输出文件被占用

原因：Excel 或 Word 文件正在打开。

处理：

1. 先关闭对应文件。
2. 再运行一次。
3. 程序也可能自动生成带时间戳的新文件名。

### 3. 个别发票识别不准

原因：

1. 发票版式特殊。
2. OCR 文字有错。
3. 图片质量过低。

处理：

1. 优先使用原始电子发票 PDF。
2. 尽量避免截图转 PDF。
3. 先看 Excel 明细，手工修正极少数异常项。

### 4. 运行时提示缺少模块

常见报错：No module named xxx。

处理：

```bash
pip install pandas pdfplumber pymupdf python-docx openpyxl
```

## 给非技术同事的推荐工作流

1. 把发票放进一个专用文件夹。
2. 运行程序一次，自动产出 Excel + Word。
3. 先看 Excel，确认金额和日期。
4. 再打印 Word 粘贴版。
5. 最后把原 PDF、Excel、Word 一起归档，避免后续补资料找不到。

---

## 注意事项

1. 本工具面向常见单页电子发票格式，极少数特殊版式可能需要人工复核。
2. 程序不会替你提交报销系统，只负责整理与排版。
3. 请对输出结果做最终人工确认，尤其是金额和日期。
4. 建议每次报销单独建一个文件夹，避免混入历史发票。
